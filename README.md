# K-means-and-SVM
K-means聚类与SVM分类对比
1. K-means 聚类
K-means 是一种迭代的无监督聚类算法，用于将 N 个数据点划分为 K 个簇。
● 迭代过程： 随机初始化 K 个中心点 μk；重复以下两步直至收敛：
1. 指派步： 将每个点指派给最近的簇中心。
2. 更新步： 重新计算每个簇的中心点（即簇内所有点的均值）。
● 评估： 由于没有真实标签，聚类结果常通过轮廓系数（Silhouette Score）或误差平方和（SSE）（用于肘部法则）进行内部评估。

2. 支持向量机（SVM）
SVM 是一种强大的有监督分类模型，旨在找到一个最大化间隔的超平面，以区分不同类别的数据点。
● 最大间隔： 确保分类平面离最近的训练样本（支持向量）有最大的距离。
● 核技巧： 当数据线性不可分时，通过核函数（如径向基函数 RBF）将数据映射到高维特征空间，使其在高维空间中变得线性可分。
